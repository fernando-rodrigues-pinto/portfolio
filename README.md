## 🛠 Technical Highlights

This project was architected to demonstrate senior-level concepts even at an early career stage:

- **Vue 3 (Composition API):** Ensuring a scalable and type-safe codebase.
- **Custom i18n System:** A fully functional internationalization engine (PT/EN) managed via global state (Pinia), proving my ability to handle complex application states.
- **Clean Tech UI:** A minimalist, high-performance interface built with **Tailwind CSS**, featuring a native Dark/Light mode toggle.
- **Responsive Architecture:** Mobile-first approach, ensuring a seamless experience across all devices.

## 🤖 AI & Development Ethics

I believe the future of software engineering lies in the orchestration of tools. This portfolio was built using **AI as a catalyst for productivity**, not as a crutch for logic. 

Every line of code generated was technically audited, refactored, and integrated by me to ensure it meets high-performance standards and maintainable architecture. I value the efficiency of AI for boilerplate while maintaining full human control over UX strategy and core logic.
